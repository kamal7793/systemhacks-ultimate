LDAP - Directory Services

Avancé

Administrez vos services d'annuaire LDAP pour l'authentification centralisée.

  • ldapsearch -x -H ldap://localhost -b "dc=example,dc=com" - Recherche LDAP
  • ldapadd -x -D "cn=admin,dc=example,dc=com" -W -f user.ldif - Ajouter entrée
  • ldapmodify -x -D "cn=admin,dc=example,dc=com" -W -f modify.ldif - Modifier entrée
  • ldapdelete -x -D "cn=admin,dc=example,dc=com" -W "uid=user,ou=people,dc=example,dc=com" - Supprimer
  • slapcat > backup.ldif - Sauvegarde OpenLDAP
  • systemctl status slapd - Status serveur LDAP
  • tail -f /var/log/slapd.log - Logs LDAP
